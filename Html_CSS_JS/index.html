Por isso, em minha primeira página html eu, não quis quebrar a tradição.

Fundamentos da Web: O HTML é a base da programação (como se fosse o esqueleto o corpo), o CSS é o que dá estilo aos códigos (como se fosse a roupa que a pessoa utiliza) e o Javascript é o que dá movimento e ação (como se fosse o espírito do corpo).
